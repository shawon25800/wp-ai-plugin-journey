A beautiful and powerful wine review plugin with star ratings, comments, average calculation, and full admin control.

== Description ==

AI Wine Rater allows users to rate wines with stars and leave comments. Perfect for wine blogs, food sites, or review platforms.

**Features:**
- Custom Post Type "Wines"
- Frontend star rating with optional comments
- Average rating with half-star support
- Glassmorphism styled settings page
- Admin can view and delete individual reviews
- Responsive and modern design

Built with love while learning WordPress plugin development with Grok AI üç∑

== Installation ==

1. Upload the `ai-wine-rater` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Wine Rater" menu to configure settings
4. Create "Wine" posts and start collecting reviews!
